Processing Student Loan Data with Python & Pandas

December 2021

An initial venture into the world of data analytics, this project utilized Python and the Pandas library to obtain and parse the information necessary to derive information from the dataset. The dataset in question is from the U.S. Government's report on Federal Loan Disbursements for the AY 2019. The data was categorized by program of study and provided information regarding the total amount of money disbursed as well as the number of recipients who received the funds. The initial thought behind the project was to develop a visual allowing me to see an estimate of how much money was being disbursed to a given number of students.

The data was imported into python using the pandas library by reading a CSV file containing the data. The data had headers for program of study, amount disbursed, and number of total disbursements (or students receiving the funds.) The data could not be extracted as-is, due to the fact that the data was formatted for readability. This included unusual formatting and extra characters such as $ for the dollar amounts and commas separating the thousands in both students and disbursement amounts. Before the data could be analyzed, unnecessary characters were stripped via regex and list iterations until a useable, anlyzeable set of data was derived. From this point, matplotlib was used to plot a scatter chart with amounts disbursed on the y-axis and students on the x-axis.
